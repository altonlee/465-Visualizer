About Visualizer 465

Visualizer 465 is a Final Project for ICS 465.

Motivations

The main goal of the app is to be a lightweight, form-driven data visualizer web application. There are not many data visualizing apps or tools available online, and while Excel and Google Sheets exist, my app exists solely to keep things simple, and to cut down the options so the user can get to sharing data more simply.

Visualizer 465 is intended to bring data to life in a visually appealing interface. As a lightweight and cheap project, this application uses JQuery, Semantic UI, and the Highcharts API.
